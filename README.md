# 産め！こうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* こうかとんが左右に動きながら敵を倒すシューティングゲーム
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 左右の矢印キーでこうかとんを操作し，スペースキー押下で卵を発射する
* ライフが０になるとゲームオーバー

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 敵キャラクター1体の追加
* 卵の発射

### 分担追加機能
* チャージ攻撃（担当：）：卵を発射していないとき、一定時間ごとに卵の威力が上がっていく
* アイテム追加（担当：）：獲得すると特殊な効果を発揮するアイテムの追加
* 雑魚敵追加（担当：）：敵の種類増加
* ボス追加（担当：）：条件達成時に現れるボスの追加
* 効果音追加（担当：）様々な効果音の追加
